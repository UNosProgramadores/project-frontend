# 🚀 Sobre el Proyecto

**ParKing Frontend** es la aplicación web encargada de proporcionar la
interfaz gráfica del sistema **ParKing**, permitiendo a administradores
y usuarios interactuar de forma sencilla con todas las funcionalidades
del sistema de gestión de parqueaderos.

La aplicación consume los servicios del backend mediante una API REST,
mostrando información en tiempo real sobre la disponibilidad de espacios,
el registro de vehículos, la administración de tarifas y la gestión de
las operaciones del parqueadero.

---

## ✨ Funcionalidades

- 🚗 Registro de ingreso y salida de vehículos.
- 🅿️ Visualización gráfica del estado del parqueadero.
- 📊 Consulta de disponibilidad en tiempo real.
- 💰 Gestión de tarifas y cálculo automático de cobros.
- 👤 Administración de usuarios y autenticación.
- 📈 Visualización de reportes y estadísticas.
- 📱 Interfaz moderna, intuitiva y responsiva.

## ⚙️ Instalación

Clonar el repositorio

```bash
git clone https://github.com/UNosProgramadores/project-frontend.git
```

Entrar al proyecto

```bash
cd project-frontend
```

Instalar dependencias

```bash
npm install
```

Ejecutar el servidor de desarrollo

```bash
npm run dev
```

La aplicación estará disponible en:

```
http://localhost:5173
```

## 📌 Objetivo

Desarrollar una interfaz web moderna e intuitiva que permita interactuar
con el sistema ParKing, facilitando la administración del parqueadero,
la visualización del estado de ocupación, el registro de operaciones y
la consulta de información en tiempo real mediante una experiencia de
usuario clara y eficiente.
